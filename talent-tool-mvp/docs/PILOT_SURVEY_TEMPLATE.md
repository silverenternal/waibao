# Pilot 调查问卷模板

> T1106 — 用于深度访谈与问卷的标准模板。PM 和 Researcher 都可以直接套用。

---

## A. 入门问卷 (第 1 天,在线填写)

收集用户基本画像与期望。

1. 您的角色是?
   - [ ] 招聘方 (HR / Recruiter)
   - [ ] 用人经理 (Hiring Manager)
   - [ ] CEO / 创始人
   - [ ] 其他: __________

2. 您团队当前最大的招聘痛点是? (多选)
   - [ ] 简历筛选耗时
   - [ ] 候选人匹配不准
   - [ ] 沟通效率低
   - [ ] 数据追踪困难
   - [ ] 协作 / 信息同步
   - [ ] 其他: __________

3. 您目前使用的工具?
   - [ ] LinkedIn Recruiter
   - [ ] Boss直聘 / Lagou
   - [ ] 内部 ATS (名称: ___)
   - [ ] Excel / 邮件
   - [ ] 其他: __________

4. 过去一个月主动联系的候选人数量?
   - [ ] 0-10
   - [ ] 11-30
   - [ ] 31-100
   - [ ] 100+

5. 试用 waibao 的主要期望? (开放题)

---

## B. 中期问卷 (第 14 天,在线填写)

- 通过 `QuickSurvey` 弹窗 (3 题 1-5 分) 收集定量数据。
- 同时附 1 道开放题。

### B.1 定量题 (1=很差, 5=很棒)

1. **易用性**: 界面是否清晰、操作是否顺手?
2. **价值感**: 是否真正帮您节省时间 / 找到合适的机会?
3. **响应速度**: 匹配 / 加载 / 反馈是否及时?

### B.2 开放题

4. 过去两周,您印象最深的一次体验是?(可正可负)

5. 您还希望 waibao 增加什么功能?

---

## C. NPS 评分 (第 7 / 14 / 28 天,在线填写)

经典 NPS 单题:

> **您向朋友推荐 waibao 的可能性有多大?**
> (0 = 完全不会推荐,10 = 一定会推荐)

跟进题 (选填):

- 为什么给这个分数?
- 哪些功能让您觉得满意 / 不满意?

---

## D. 深度访谈提纲 (30-45 min)

### D.1 开场 (5 min)

- 您能简单介绍一下您的工作和团队吗?
- 您目前在招聘方面的最大挑战是什么?

### D.2 体验回顾 (15 min)

- 试用过程中,哪些功能让您觉得"惊喜"?
- 哪些功能让您觉得"失望"或"困惑"?
- 您能具体描述一次糟糕的体验吗? (触发场景、操作步骤、结果)
- 您能具体描述一次超出预期的体验吗?

### D.3 匹配质量 (5 min)

- 推荐的候选人里,前 5 名您认为靠谱的有几位?
- 推荐的解释 (reasoning trace) 是否清晰、可信?
- 您觉得哪些信息应该出现在解释里但目前缺失?

### D.4 协作体验 (5 min)

- 您是否尝试过协作房间?
- 如果用过,您邀请了谁? 协作流程是否顺畅?
- 如果没用过,为什么?

### D.5 商业价值 (5 min)

- 您估算 waibao 每周为您节省了多少时间?
- 如果续费,您愿意支付的范围?
- 哪些因素会影响您的续费决策?

### D.6 收尾 (5 min)

- 您还有什么想分享的?
- 您愿意把 waibao 推荐给同行吗? 为什么?

---

## E. 退出问卷 (试用结束后)

1. 您是否会续费?
   - [ ] 是
   - [ ] 否
   - [ ] 还在评估

2. (选否) 阻碍您续费的最大因素?
   - [ ] 价格
   - [ ] 功能不满足需求
   - [ ] 使用习惯
   - [ ] 团队协作问题
   - [ ] 其他: __________

3. 您会推荐给同行吗?(NPS)

4. 任何额外建议: __________

---

## F. 数据录入规范

- 问卷结果 PM 在 24h 内录入 CRM / Notion。
- 反馈录入 `pilot_feedback` 表 (category=`survey` 或 `nps`)。
- 访谈录音 PM 自行决定是否存档,默认不外传。
- 个人身份信息脱敏后再用于报告。
