## 一、求职者手册

> 你是找工作的候选人。系统会帮你整理简历、和 AI 聊职业、生成专属画像、规划职业路径,并把你推荐给合适的企业。

### 1.1 注册与登录
1. 打开首页 → 点“求职者入口 / 注册”;
2. 填手机号/邮箱 + 密码(或验证码)→ 提交;
3. 登录后进入“我的主页”(Dashboard)。

### 1.2 上传与管理简历
1. 进入“我的简历”;
2. 点“上传简历”,选择 PDF / 图片 / 文本;
3. 系统会用**本地 OCR** 自动识别简历内容,提取姓名、技能、学历、工作经历;
4. 核对/修正识别结果 → 保存。可上传多版本,设一份为“主简历”。

> 提示:OCR 在本地完成,简历图片不会上传到任何外部网站。

### 1.3 与 AI 聊天
1. 进入“AI 聊天”;
2. 直接提问,例如:
   - “根据我的经历,我适合体育用品行业哪些岗位?”
   - “我的简历还有哪些可以改进的地方?”
3. AI 会结合你的简历与画像给出建议。

> 如果你在聊天中感到焦虑或低落,AI 会主动关心你。如果你遇到紧急情绪困扰或劳动权益问题,系统会自动转接人工,由真人协助你。

### 1.4 生成与确认人才画像
1. 进入“人才画像” → 点“生成画像”;
2. 系统综合简历 + 对话,生成:
   - 技能标签(如:电商运营、数据分析)
   - 核心优势
   - 推荐发展方向
3. 你可以**确认**或**修正**每一条,修正后系统会更懂你。

### 1.5 职业规划与行动追踪
1. 进入“职业规划” → 点“生成规划”;
2. 获取分阶段的职业目标与行动项;
3. 在“行动项”里勾选完成情况,系统会追踪你的进度并适时提醒。

### 1.6 查看推荐岗位与机会
- “推荐岗位”会根据你的画像主动推荐合适的体育用品行业岗位;
- 感兴趣可“表达意向”,HR 会看到并可能联系你。

## 二、HR 手册

> 你是企业招聘负责人。系统帮你管理企业、发布岗位、找到匹配候选人、联系并安排面试。

### 2.1 注册企业与登录
1. 首页 → “企业入口 / 注册”;
2. 填公司名、行业、规模 → 注册 → 登录企业管理后台。

### 2.2 上传企业资质
1. 进入“企业资质”;
2. 上传营业执照等资质图片 → 系统**本地 OCR** 识别 + AI 真实性初筛;
3. 查看审核状态;被标记为“待复核”的资质需补充说明。

### 2.3 创建与管理岗位
1. 进入“岗位管理” → “新建岗位”;
2. 填写:标题、描述、技能要求(必选/优选)、薪资范围、地点、远程政策;
3. 点“AI 细化 JD” → 让 AI 把描述改得更专业、更有吸引力 → 一键采纳;
4. 保存后岗位进入“招聘中”状态。

### 2.4 查看匹配候选人
1. 进入某岗位 → “推荐候选人”;
2. 按综合匹配分排序,每位候选人展示:
   - 匹配分数 + 技能重叠 + 优势/差距 + AI 解释;
3. 点开查看脱敏画像与详情。

### 2.5 联系候选人与安排面试
1. 在候选人卡片点“发起联系” → 填写联系内容 → 系统记录**联系日志**;
2. 点“安排面试” → 选时间/方式 → 生成**面试日程**;
3. 所有联系与面试记录可在“联系日志 / 面试日程”中追溯。

### 2.6 智能推荐
- “智能推荐”位会主动给你推荐人才与岗位,帮助你发现机会。

### 2.7 HR 助手(进阶)
- 勾选 2-3 位候选人 → “对比” → 看对比表;
- “生成报告” → 输出评估报告;
- “生成面试题” → 按岗位/候选人给出针对性面试问题。

### 2.8 个性化沟通
- HR 助手会学习你们企业以往的沟通语气,给出风格一致的回复建议,可直接采纳或微调。

## 三、老板 / 管理层手册

> 你是企业决策者。系统帮你把公司愿景、战略、人才标准沉淀下来,并贯穿到招聘全流程。

### 3.1 愿景与战略录入
1. 进入“企业愿景 / 战略”模块;
2. 录入公司愿景、阶段战略、人才战略方向;
3. 保存后,这些信息会成为 AI 与 HR 行动的“北极星”。

### 3.2 战略传达
- 系统会把战略自动**传达**到岗位 JD、候选人沟通、HR 话术中,确保招聘动作与战略一致;
- 你可在“战略传达”看板查看落地情况。

### 3.3 人才框架
1. 进入“人才框架”;
2. 定义关键岗位的能力模型、人才标准、梯队结构;
3. 框架会指导匹配与推荐(避免“招进来却不对路”)。

### 3.4 数据看板
- “管理看板”展示:招聘漏斗、命中率、到岗率、人才池规模、各部门进度;
- “匹配质量”看板展示命中率与 HR 反馈率,用于评估招聘效能。

### 3.5 偏见纠正
- 系统对 JD 与评价中的潜在偏见(性别/年龄/地域等)给出纠正建议,保障公平合规。

## 四、部门负责人手册

> 你是用人部门的负责人。你参与本部门岗位的细化、候选人的评估与多方协作决策。

### 4.1 细化本部门岗位
1. 在本部门岗位列表中选择岗位;
2. 补充**岗位细节**:具体职责、技术栈、团队情况、考核标准;
3. 可点“AI 细化”让 AI 协助润色。

### 4.2 评估候选人
1. 在岗位的候选人列表中查看推荐人选;
2. 查看匹配分、技能重叠、优势/差距;
3. 给出你的反馈(赞同推进 / 保留意见 / 不合适)。

### 4.3 多方协作房间
1. 加入围绕某候选人的**协作房间**(通常含你 + HR + 候选人);
2. 在房间内实时沟通、留言、@相关人;
3. 房间内展示**共识度评分**,随各方反馈更新。

### 4.4 协同决策
- 用“对比”功能横向比较多位候选人;
- 达成共识后,在房间内确认推进,HR 据此安排后续联系/面试。

### 4.5 沉默提醒
- 若某方长时间未响应,系统会自动提醒,避免流程卡顿。

## 五、通用:账号与隐私

### 5.1 账号安全
- 首次登录后请修改默认密码;
- 不同角色(求职者 / HR / 老板 / 部门负责人)权限不同,请勿越权操作。

### 5.2 你的数据权利(GDPR / PIPL)
- **导出**:可在“隐私中心”申请导出你的全部个人数据;
- **更正**:可申请更正不准确的信息;
- **删除**:可申请删除你的账号及相关数据(依法保留的除外)。
- 敏感信息(身份证、手机、薪资等)在数据库中**加密存储**。

### 5.3 数据不出甲方环境
- 所有简历 / 资质 / 对话均在甲方本地服务器处理;
- 大模型(Ollama)与 OCR(PaddleOCR)都是本地部署的服务,你的数据**不会被发送到任何外部公司**。

### 5.4 遇到问题
- 求职者情绪困扰 / 劳动争议 → 系统会自动转人工,你也可以主动点击“转人工”;
- 技术故障 → 联系甲方系统管理员,或参考《本地部署指南》`docs/LOCAL_DEPLOYMENT.md` 的“常见问题”。
