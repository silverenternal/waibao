# 隐私政策 (Privacy Policy)

**版本**: v1.0  
**生效日期**: 2026-07-12  
**适用产品**: waibao (外宝) — AI 招聘协同平台  
**运营主体**: Waibao Technology Co., Ltd.

---

本政策依据《中华人民共和国个人信息保护法》《数据安全法》《网络安全法》及欧盟 GDPR 制定,旨在以透明、清晰的方式说明我们如何收集、使用、存储、共享和保护您的个人信息。请仔细阅读,使用我们的服务即视为您已知悉本政策全部内容。

## 一、我们收集哪些信息

### 1.1 您主动提供的信息
- **账户信息**:姓名、邮箱、手机号、密码哈希、所在城市、职位头衔
- **候选人信息**:候选人姓名、邮箱、手机、身份证号(选填)、简历正文、工作经历、教育背景、技能标签
- **企业信息**:统一社会信用代码、企业名称、联系人、营业执照扫描件、发票抬头
- **协作内容**:多人房间中的文字、图片、文件、语音转写、决策记录

### 1.2 我们在您使用过程中自动收集的信息
- **设备与日志**:IP 地址(经哈希后存储)、设备型号、浏览器 UA、操作系统
- **行为数据**:访问页面、停留时长、功能使用频次(匿名化)
- **Cookie 与同类技术**:用于会话保持、偏好记忆、统计(详见《Cookie 政策》)

### 1.3 我们从第三方获得的信息
- 第三方招聘渠道(Boss 直聘、拉勾、Adzuna 等)经用户授权后获取的公开简历
- 钉钉 / 企业微信 / 飞书 OAuth 实名认证信息(仅当用户主动授权)

## 二、我们如何使用信息

我们仅在以下目的范围内处理您的个人信息:

| 处理目的 | 涉及字段 | 法律依据 |
|---|---|---|
| 账户注册与登录 | 邮箱、手机、密码哈希 | 合同必要 |
| 候选人画像与匹配 | 简历、技能、工作经历 | 用户明示同意 |
| 多人协作(房间、消息) | 上传内容、聊天记录 | 用户明示同意 |
| 实名认证(中国大陆) | 姓名、身份证号 | 法定义务 |
| 收费与开票 | 银行账号(脱敏存储) | 合同必要 |
| 安全审计与合规 | IP(哈希)、操作日志 | 法定职责 |
| 产品改进(脱敏统计) | 行为数据(匿名化) | 合法权益 |
| 跨境传输(可选) | 用户明示同意字段 | 用户明示同意 |

## 三、信息存储与跨境传输

### 3.1 存储位置
- **中国大陆用户**:数据默认存储在境内的阿里云 / 腾讯云数据中心,通过等保 2.0 三级测评
- **海外用户**:根据所在区域路由到对应 Supabase 实例(SG / US)
- 用户可随时申请导出个人数据副本

### 3.2 跨境传输
- 仅在以下场景进行跨境传输:
  - 您主动使用海外功能(如海外职位匹配)
  - 您明确同意跨境传输(在 Cookie 横幅中可单独选择)
- 跨境传输前会进行个人信息保护影响评估(PIPIA),并签署标准合同或取得认证

### 3.3 保留期限
| 数据类别 | 保留期限 | 删除方式 |
|---|---|---|
| 账户基本信息 | 账户存续期间 + 注销后 30 天 | 软删除 + 物理删除 |
| 简历、协作内容 | 24 个月(到期自动归档) | 物理删除 |
| 操作日志、审计 | 不少于 6 个月(金融等关键岗位 5 年) | 物理删除 |
| 实名认证信息 | 仅用于实名核验,完成后 24 小时内删除原始证件影像 | 物理删除 |

## 四、信息共享与披露

我们承诺不会出售您的个人信息。仅在以下情况共享:

1. **经您明示同意**:例如您主动分享简历给某企业 HR
2. **与关联服务方**:仅提供必要范围,且与其签署严格的数据处理协议(DPA)
3. **与监管部门**:依据法律法规或主管部门合法要求
4. **与争议解决相关**:在争议调解、诉讼、仲裁中按法定程序披露

## 五、您的权利

依据《个人信息保护法》和 GDPR,您享有以下权利:

| 权利 | 行使方式 |
|---|---|
| **知情权** | 本政策 + 数据处理清单 |
| **访问权** | GET /api/gdpr/export |
| **更正权** | 各类资源 PATCH 接口 |
| **删除权(被遗忘权)** | DELETE /api/gdpr/all-data |
| **可携权** | GET /api/gdpr/export (JSON / CSV) |
| **撤回同意** | 账户 → 隐私设置 |
| **拒绝自动化决策** | 账户 → 隐私设置(关闭 AI 辅助匹配) |
| **投诉与申诉** | 联系 DPO / 网信办 / 当地监管 |

我们将在 15 个工作日内响应您的请求,特殊情况不超过 30 天。

## 六、信息安全

- **加密**:PII 字段级 Fernet(AES-128-CBC + HMAC-SHA256);敏感文件存储加密
- **访问控制**:基于角色的最小权限模型,所有访问需 MFA
- **审计**:所有 PII 访问均记录到 audit_log,保留不少于 6 个月
- **渗透测试**:每季度一次第三方渗透测试
- **应急响应**:发现安全事件后 24 小时内通知用户和监管

## 七、未成年人保护

本平台不面向 14 周岁以下未成年人。如发现,我们将主动删除相关数据。

## 八、政策变更

我们可能根据业务或法律变更调整本政策。重大变更会通过站内通知或邮件告知。变更后的政策自发布之日起生效。
