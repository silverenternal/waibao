## 3. Demo 流程 (30 分钟)

| 分钟 | 环节 | 关键内容 |
|---|---|---|
| 0-5 | 痛点诊断 | 询问当前 ATS、月活招聘官数、漏斗转化率 |
| 5-15 | Mothership 后台演示 | AI 匹配 Top-5 候选人 / 协同房间 / 看板 |

| 25-30 | 试用条款 + 下一步 | 30 天免费 / 5 用户 / CSM 1v1 / 续费优惠 30% |

## 6. FAQ

**Q: 我们已有 ATS, 还要用 waibao?**
A: 我们有 ATS 集成 (Greenhouse / Lever / 北森 / Beisen),数据双向同步,不替代。

**Q: 数据安全?**
A: GDPR + 中国《个人信息保护法》双合规,UK/EU/CN/SG 多区域可选,SSO/SAML。

**Q: 试用结束一定要付费?**
