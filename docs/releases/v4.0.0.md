# waibao v4.0.0 — Production Release

> **Released**: 2026-07-12 · **Code name**: "Global Ready"
> 16 Agents · 1242 Tests · 3 Regions · 4 Platforms · 28 Providers

---

## 🎯 Highlights

### Global Multi-Region Deployment
- **region-cn** (阿里云 cn-hangzhou) — ICP 备案, MLPS 2.0 合规, 钉钉/微信集成
- **region-sg** (AWS ap-southeast-1) — GDPR/PDPA 合规, ATS 同步, 视频面试
- **region-us** (AWS us-west-1) — CCPA/SOC 2, Stripe/Checkr, 北美主库
- **GeoDNS** — alidns (国内) + Cloudflare (海外) 智能路由
- **RTO ≤ 15 min / RPO ≤ 5 min / SLA 99.9%**

### 4 端覆盖 (Multi-Platform)
- **Web** — Next.js 16 (SSR + 响应式)
- **微信小程序** — uni-app 跨端
- **钉钉微应用** — 完整 corp_binding + 审批
- **飞书应用** — 同上
- **PWA** — Service Worker + 离线 fallback

### 9 项 AI 深度能力
1. AI 自动面试 (出题 → 答题 → 评分 → 报告)
2. Offer 比较器 (多 Offer + 汇率/税/折现)
3. 谈判建议 (行业基准 + LLM 脚本)
4. 招聘漏斗 (5 阶段转化率)
5. 渠道 ROI (cost / hire)
6. 候选人订阅 (关键词 + 地点 + 薪资)
7. AI 推荐 (embedding-based)
8. 全局搜索 ⌘K (跨实体)
9. 测评 + 背调 (北森 + Checkr)

### 5 个新供应商
- **支付**: Stripe / 微信支付 / 支付宝
- **视频**: Zoom / 腾讯会议
- **ATS**: Greenhouse / Lever
- **测评**: 北森 (Beisen)
- **背调**: Checkr

### 合规 (3 法规)
- **PIPL** (中国) — PII 字段级 AES-GCM, 数据不出境
- **GDPR** (EU) — 被遗忘 + 数据可携 + cookie banner
- **CCPA** (CA) — opt-out + 数据导出

### 可访问性
- WCAG 2.1 AA — SkipToMain / 高对比度 / 键盘导航 / 减少动效

## 📦 升级步骤

```bash
# 1. 拉新代码
git pull origin master

# 2. 安装新依赖
cd backend && pip install -r requirements.txt
cd ../frontend && npm install

# 3. 数据库迁移
psql $DATABASE_URL -f supabase/migrations/019_pilot_programs.sql
psql $DATABASE_URL -f supabase/migrations/020_pii_encryption_keys.sql
psql $DATABASE_URL -f supabase/migrations/021_third_party_corp.sql
# ... 一直到 029_ats_sync.sql

# 4. 环境变量 (新增 v4.0 字段)
# - REGION / DATA_RESIDENCY / PII_ENCRYPTION_KEY
# - DEEPSEEK_API_KEY / QWEN_API_KEY / STRIPE_SECRET_KEY
# - ZOOM_API_KEY / GREENHOUSE_API_KEY / CHECKR_API_KEY / OSS_*

# 5. 重启服务
docker-compose -f docker-compose.prod.yml up -d --build
# 或 K8s
kubectl apply -k infra/region-{cn,sg,us}/k8s/
```

详见 [DEPLOYMENT.md](DEPLOYMENT.md) / [MULTI_REGION.md](MULTI_REGION.md)
