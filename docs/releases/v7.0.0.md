# waibao v7.0.0 — Enterprise SaaS · Deep AI

> **发布日期**: 2026-07-13 · **代号**: "SaaS-Ready, AI-Deep"
> **上一版**: [v6.0.0](./v5.0.0.md) (2026-01-15) · **覆盖**: 16 → 32 需求 / 2273 → **2332 collected** tests (含 84+ P0 合规/审计 / GDPR / SLA / Status test)
> **配套**: [COMMERCIAL_LICENSE.md](../COMMERCIAL_LICENSE.md) · [SUBSCRIPTION_PLANS.md](../SUBSCRIPTION_PLANS.md) · [SLA.md](../SLA.md) · [STATUS_PAGE.md](../STATUS_PAGE.md)

v7.0 是 waibao 的第一个 **真正能商业化的发行版**。它在 v6.0 (16/16 需求 + 2273 tests + 可扩展架构 + 差异化 AI) 基础上,完成两件大事:

1. **Enterprise SaaS 化** — 让产品可以卖给大客户:严格多租户 + 配额 + 限流 + 完整审计 + GDPR/PIPL/CCPA + SLA 99.9% + 状态页 + 自助订阅
2. **AI 能力深化** — 真正的技术护城河:完整 RAG + 统一 Agent Memory + Multi-Agent 协作框架 + Prompt 版本化与自动评估 + 多模态语义搜索

---

## 一行 TL;DR

> **v6 是产品,v7 是 SaaS + AI 平台。** 自此 waibao 可直接上架销售、签署 MSA、签发 SLA、面向 Enterprise 投标、对外宣传"SOC 2 / ISO 27001 / GDPR 全合规"。

## 2. P0 — Enterprise SaaS 化 (5 周 / 4 主任务)

### T2601 — 严格多租户隔离 (RLS + Tenant 上下文)

- Postgres RLS policies on **所有 47 张业务表**
- `services/platform/tenant_context.py` — 全局 Tenant 上下文 (per-request)
- `services/platform/tenant_resolver.py` — 域名 / JWT / API Key 三路解析
- `audit_log_v2.tenant_id` 强制 NOT NULL + RLS
- 跨租户访问返回 `403 cross-tenant-forbidden` (审计记录)

> **测试**: RLS 覆盖 + 跨租户访问 422 用例,集成到 `tests/test_tenant_isolation.py`

### T2602 — 统一 Rate Limiting + 配额管理

- **slowapi** + Redis 集中桶 (`services/platform/rate_limiter.py`)
- 全局 5/30/200 RPS × Starter / Growth / Enterprise
- 关键资源 per-minute: candidates / match / llm/chat / agents/run / uploads / webhooks / gdpr-forget
- 超额 → HTTP 429 + `Retry-After` + `X-RateLimit-*` 标准头
- 配额使用情况实时展示 (Billing Portal)

### T2603 — 完整审计日志 + GDPR/PIPL/CCPA 合规

- **Audit v2** 迁移: `audit_log_v2` 表 (per-purpose · immutable · append-only)
- `services/platform/audit_v2.py` 服务 + **AST 装饰器** 自动埋点
- 同意矩阵升级: per-purpose (营销 / 推荐 / 第三方共享 / 跨境 / 训练)
- **GDPR v2 API**:
  - `POST /api/v1/gdpr/forget` — 级联删 OLTP + pgvector + ClickHouse + Redis + Storage
  - `GET /api/v1/gdpr/export` — JSON + CSV + parquet
  - `POST /api/v1/gdpr/rectify` — 数据更正
- 前端隐私偏好中心 (per-purpose toggle)

> **测试**: `tests/test_audit_v2.py` + `tests/test_gdpr_v2.py` + `tests/test_consent.py` + `compliance/tests/test_consent.py` — **84 tests 全 PASS**

### T2604 — SLA 99.9% + 状态页 + 客户支持

- `services/platform/sla_monitor.py` — 5 服务监控 (api / llm / storage / webhook / database)
- `GET /api/v1/sla/uptime` + `/api/v1/sla/incidents` + `/api/v1/sla/credits`
- **自托管 Instatus** 状态页 (Caddy + Docker)
- **Intercom** 在线客服 + 前端 `SupportWidget`
- 钉钉 / 飞书 / PagerDuty / Webhook 4 类 incident 通知
- 现场审计: SOC 2 Type II + ISO 27001 + 渗透测试报告 (年度)

> **测试**: `tests/test_sla_monitor.py` + `tests/test_status_page.py` + `tests/test_alerting.py` — 全部通过

## 3. P1 — AI 能力深化 (6 周 / 4 主任务)

### T2701 — 完整 RAG (Document Parser + Chunking + Rerank + Citation)

- `services/rag/document_parser.py` — PDF / DOCX / PPTX / HTML / Markdown 统一解析
- `services/rag/chunker.py` — 语义分块 + sliding-window + metadata 标注
- `services/rag/reranker.py` — cross-encoder (bge-reranker-large) 重排
- `services/rag/citation.py` — 答案引用溯源 + 页码 / 段落精确定位
- pgvector (1024 维) + HNSW 索引 + IVFFlat 备选

### T2702 — 统一 Agent Memory + 跨会话

- `services/agents/memory/` — 短期 (Redis) + 长期 (pgvector + JSONB) 双层存储
- 跨 session 召回 (similarity > 0.78) + 自动压缩 + 隐私脱敏
- TTL 可配 (Starter 30d / Growth 6mo / Enterprise custom)

### T2703 — Multi-Agent 协作框架

- `services/agents/orchestration/` — DAG 调度 + MessageBus + Retry + Compensation
- 5 内置 workflow 模板:**Sourcing Pipeline** · **Interview Loop** · **Offer Negotiation** · **Onboarding** · **Re-engagement**
- 自定义 workflow (React Flow 可视化): `[id]/page.tsx`
- `036_agent_workflows.sql` 持久化 + versioning

### T2704 — Prompt 版本化 + 自动评估

- `services/prompts/registry.py` — versioned prompts + 灰度发布 + 回滚
- `services/prompts/evaluator.py` — 离线评估集 (200+ ground truth) + LLM-as-judge + 人评抽样
- A/B 实验 + 实时指标 → Config Center 自动调优

### T2501 — 多模态语义搜索 (进行中)

- 文本 + 图像 (CLIP-ViT-L/14) + 音频 (Whisper large-v3) + 视频 (VideoLLaMA)
- 跨模态统一向量空间 + 多模态 query

## 8. 迁移指南 (Migration from v6.0)

### 8.1 自助迁移 (Starter / Growth)

- **无操作** — 数据库 + RLS + 审计已自动就位
- 现有 API Key + Integration 100% 兼容
- 推荐:开启 SSO (Google / Microsoft) · 安装 1 个新审计 dashboard widget

### 8.2 Enterprise 客户

1. **法务**: 签署新的 MSA + DPA + SLA Annex(模板见 `COMMERCIAL_LICENSE.md`)
2. **安全**: 启用 SSO/SAML + SCIM · 替换密码登录
3. **数据**: 选择数据驻留区域 (CN/HK/SG/EU) · 决定加密 (BYOK 或 waibao-managed)
4. **集成**: 配置 mTLS Webhook · 升级 ATS adapter 到 v7 RLS-aware
5. **运维**: 配置 PagerDuty · 订阅状态页 webhook · 启用 QBR + TAM

### 8.3 自托管 (Self-Hosted) 客户

- 邮件 `gtm@waibao.example.com` 申请 v7 二进制 + 许可激活
- 部署脚本: `infra/scripts/install_v7.sh` (Docker Compose + Helm chart)
- 升级窗口:**2026-09-01 ~ 2026-12-31**(6 个月并行运行 v6.0 / v7.0)

### 8.4 破坏性变更 (Breaking Changes)

| API | 变更 | 迁移 |
|-----|------|------|
| `audit_log` 表 | **已弃用**,仅审计 v1 兼容 1 个月 | 切到 `audit_log_v2` |
| `/api/v1/billing/invoices` 响应 | 新增 `tax_id`, `currency` 字段 | 升级 SDK ≥ 7.0 |
| `LLM_PROVIDER` 环境变量 | 接受 `dashscope` 替代 `aliyun_qwen` | 重命名即可 |
| `plugin.yaml` v1 manifest | 升级到 v2 schema (新增 `permissions`) | 一行迁移 |
