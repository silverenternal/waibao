### F. 业务深度不足

虽然 16 项需求 ✅,但**业务深度**仍有空间:

**求职者侧**:
- ❌ **简历市场**(简历被猎头/HR 浏览付费?)
- ❌ **AI 自动面试**(用 GPT-4V/Whisper 模拟面试官)
- ❌ **薪资谈判助手**(面试 offer 怎么谈)
- ❌ **Offer 比较器**(多个 offer 怎么选)
- ❌ **离职预测**(从情绪时间线 + 工作状态预测离职)
- ❌ **内部推荐**(在职员工推荐候选人)

**用人单位侧**:
- ❌ **招聘漏斗分析**(从 sourcing → offer 的转化漏斗)
- ❌ **招聘渠道 ROI**(Boss直聘/拉勾/猎头哪个 ROI 高)
- ❌ **背景调查集成**(iCIMS / Checkr)
- ❌ **测评集成**(北森/光辉国际)
- ❌ **视频面试集成**(Zoom / 腾讯会议)
- ❌ **Offer 审批流**(HR → 部门负责人 → CEO)
- ❌ **ATS 双向同步**(与 Greenhouse/Lever 同步)

**双向匹配深化**:
- ❌ **候选人推荐引擎**(主动给 HR 推荐)
- ❌ **候选人订阅**(求职者订阅"AI 产品经理"职位,有新职位自动推)
- ❌ **公司评价集成**(看准网 / Glassdoor)
- ❌ **行业薪资报告**(行业 / 城市 / 职级 薪资分布)

## v4.0 优先补强方向

1. **P0 真实集成**: 至少 3 个真实第三方 API 跑通(Boss直聘/Whisper/钉钉 webhook),不停留在 mock
2. **P0 性能压测**: 用 Locust/wrk 压测核心 API + WebSocket,产出 SLA 报告
3. **P0 中型企业试用**: 找 1-2 家试用,1 个月,收集 NPS + 反馈
4. **P1 合规补齐**: ToS/Privacy/GDPR cookie banner/数据导出 UI/网络安全法等保 2.0
5. **P1 多端**: 微信小程序 + 钉钉嵌入应用(中国) / PWA(海外)
6. **P1 业务深度**: AI 自动面试 + Offer 比较 + 招聘渠道 ROI + 候选人订阅
7. **P2 生态集成**: 与 ATS 双向同步 / 视频面试 / 背景调查
8. **P2 UX**: A11y / SEO / 新手引导 / 产品内搜索
